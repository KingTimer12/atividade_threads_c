# Como usar?
Crie, manualmente, uma pasta chamada `build`<br>
Execute o script build de acordo com seu sistema operacional:
### Para Unix
```bash
bash build.sh
```
### Para Win
```pwd
.\build.ps1
```

Após os programas serem construídos, execute-os com o script run:
### Para Unix
```bash
bash run.sh <questao>
```
### Para Win
```pwd
.\run.ps1 <questao>
```
